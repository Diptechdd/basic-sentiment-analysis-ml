## Dataset Description

The dataset `social_media_reviews.csv` contains text data collected from
social media platforms and labeled with sentiment polarity.

### Columns
- text : User-generated review or comment
- sentiment : Sentiment label (e.g., positive / negative or 1 / 0)

### Purpose
This dataset is used to train and evaluate a basic sentiment analysis
model using classical machine learning techniques.

The dataset is suitable for educational and foundational research work.
